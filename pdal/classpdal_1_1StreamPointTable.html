Detailed Description

A StreamPointTable must provide storage for point data up to its capacity. It must implement getPoint() which returns a pointer to a buffer of sufficient size to contain a point's data. The minimum size required is constant and can be determined by calling pointsToBytes(1) in the finalize() method.
