99 // The FauxReader doesn't read from disk, but instead just makes up data for its
100 // points. The reader is constructed with a given bounding box and a given
101 // number of points.
102 //
103 // This reader knows about these fields (Dimensions):
104 // X,Y,Z - floats
105 // Time - uint64
106 // ReturnNumber (optional) - uint8
107 // NumberOfReturns (optional) - uint8
108 //
109 // It supports a few modes:
110 // - "random" generates points that are randomly distributed within the
111 // given bounding box
112 // - "constant" generates its points to always be at the minimum of the
113 // bounding box
114 // - "ramp" generates its points as a linear ramp from the minimum of the
115 // bbox to the maximum
116 // - "uniform" generates points that are uniformly distributed within the
117 // given bounding box
118 // - "normal" generates points that are normally distributed with a given
119 // mean and standard deviation in each of the XYZ dimensions
120 // In all these modes, however, the Time field is always set to the point
121 // number.
122 //
123 // ReturnNumber and NumberOfReturns are not included by default, but can be
124 // activated by passing a numeric value as "number_of_returns" to the
125 // reader constructor.
126 //
