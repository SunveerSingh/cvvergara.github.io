Extract clusters of points from input PointView.

For each point, find neighbors within a given tolerance (Euclidean distance). If a neighbor already belongs to another cluster, skip it. Otherwise, add it to the current cluster. Recursively visit newly added cluster points, looking for neighbors to add to the cluster.

Parameters
[in]viewthe input PointView.
[in]min_pointsthe minimum number of points in a cluster.
[in]max_pointsthe maximum number of points in a cluster.
[in]tolerancethe tolerance for adding points to a cluster.
Returns
a vector of clusters (themselves vectors of PointIds).
