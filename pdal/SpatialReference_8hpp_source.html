45 /// A SpatialReference defines a model of the earth that is used to describe
46 /// the location of points.
47 /// A SpatialReference is part of input data and is automatically loaded
48 /// into PDAL by readers, or it's provided explicitly by a user through an
49 /// option. All points in a point view share a common spatial reference. When
50 /// a stage finishes processing point view, the point view takes on the
51 /// spatial reference of that stage, if it had one.
52 /// A point table tracks the spatial references of the views currently being
53 /// processed by a stage. If a point table being processed by a stage has
54 /// more than one spatial reference, PointTable::spatialReference() will
55 /// return an empty spatial reference and PointTable::spatialReferenceUnique()
56 /// will return false.
