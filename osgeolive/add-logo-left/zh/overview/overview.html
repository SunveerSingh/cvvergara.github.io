OSGeo-Live 11.0beta 包含内容一览¶

桌面 GIS¶

用于桌面电脑环境的 GIS 信息浏览、编辑和分析工具

  • Quantum GIS (QGIS) - [快速入门] — 桌面 GIS
  • GRASS GIS - [快速入门] — 桌面 GIS
  • gvSIG 桌面系统 - [快速入门] — 桌面 GIS
  • User-friendly Desktop Internet GIS (uDig) - [快速入门] — 桌面 GIS 框架
  • Kosmo Desktop - [快速入门] — 桌面 GIS
  • OpenJUMP GIS - [快速入门] — 轻量化桌面 GIS
  • SAGA - [快速入门] — 桌面 GIS

网页 GIS（Browser Facing GIS）¶

用于浏览器环境的 GIS 信息浏览、编辑和分析工具

  • OpenLayers - [快速入门] — 网页 GIS 服务
  • Leaflet - [快速入门] - 移动设备用户友好的交互地图
  • Cesium - [快速入门] - 基于浏览器的3维地球与2维地图
  • Geomajas - [快速入门] — 网页 GIS 服务
  • Mapbender - [快速入门] — 网络地图服务集成框架
  • MapFish - [快速入门] — 网页制图服务框架
  • GeoMoose - [快速入门] — 网页 GIS 服务集成
  • Cartaro - [快速入门] - 具备空间功能的内容管理系统
  • GeoNode - [快速入门] - 地理空间内容管理系统

空间信息的网络发布

  • GeoServer - [快速入门] — 空间数据网络发布
  • MapServer - [快速入门] — 空间数据图形表达和管理
  • deegree - [快速入门] — 网络空间数据服务
  • ncWMS - [快速入门] - Web Map Service
  • EOxServer - [快速入门] - 网络地理信息编录服务
  • GeoNetwork - [快速入门] — 元数据编目管理
  • pycsw - [快速入门] — 元数据编目发布
  • Map Proxy - [快速入门] — 网页地图及切片服务代理
  • QGIS Server - [快速入门] — 网页地图服务
  • 52°North WPS - [快速入门] — 网页空间数据处理服务
  • 52°North SOS - [快速入门] — 传感器观测数据获取
  • TinyOWS - [快速入门] — 基于事务的网页空间要素服务
  • ZOO-Project - [快速入门] — 网页空间数据处理服务

地图与导航¶

  • GPS Drive - [快速入门] — GPS 导航

  • Viking - [快速入门] — GPS 数据编辑与分析

空间工具¶

空间信息分析系统

  • GeoKettle - [快速入门] — 空间信息数据库整合
  • GMT (Generic Mapping Tools) - [快速入门] — 地图制作
  • Mapnik - [快速入门] — 地图制作
  • TileMill - [快速入门] - 地图渲染和发布
  • MapSlicer - [快速入门] — 栅格地图切片化
  • OSSIM [快速入门] - 影像处理
  • OTB - [快速入门] — 图像处理
  • R 空间分析功能 - [快速入门] — 空间数据统计
  • IPython - [快速入门] — IPython交互编程环境

数据¶

地理空间数据集

  • Natural Earth — 地学基础数据集
  • OSGeo 美国北卡洛莱娜州教育用数据集 — 北卡罗莱纳州地理数据集
  • Open Street Map — Open Street Map 的数据样本
  • NetCDF数据集 - 时间序列气温和降水

地理空间操作函数库¶

  • GDAL/OGR - [快速入门] — 空间数据格式转换
  • JTS Topology Suite (JTS) 拓扑运算函数库 - Java 语言的拓扑运算
  • GeoTools - Java GIS 工具箱
  • GEOS - C/C++ 空间操作
  • Proj.4 - [快速入门] - 坐标投影转换 (MetaCRS)
  • LibLAS - [快速入门] - LiDAR 激光立体扫描数据处理
  • Iris - [快速入门] - 用于气象学与气候学数据处理的Python库
