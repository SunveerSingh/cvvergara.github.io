Contenidos OSGeo-Live 11.0beta¶

SIG de escritorio¶

SIG de uso general. Visualización, edición y análisis de Escritorio:

  • QGIS - [QuickStart]
  • GRASS GIS - [QuickStart]
  • gvSIG Desktop - [QuickStart]
  • User-friendly Desktop Internet GIS (uDig) - [QuickStart]
  • Kosmo Desktop - [QuickStart]
  • OpenJUMP GIS - [QuickStart]
  • SAGA - [QuickStart]

Clientes ligeros o de navegador Web¶

Visualización, edición y análisis en el navegador:

  • OpenLayers 3 - [QuickStart] - Cliente GIS de Navegador Web.
  • Leaflet - [QuickStart] - Mapas interactivos adaptados a móviles.
  • Cesium - [QuickStart] - 3Globos 3D y mapas 2D en el navegador.
  • Geomajas - [QuickStart] - Cliente GIS de Navegador Web.
  • Mapbender - [QuickStart] - Framework de Geoportal .
  • GeoMoose - [QuickStart] - Portal web SIG.
  • Cartaro - [QuickStart] -CMS (Sistema de Gestión de Contenidos) Geoespacial.
  • GeoNode - [QuickStart] - CMS (Sistema de Gestión de Contenidos) Geoespacial.

Publicación de datos espaciales en internet:

  • GeoServer - [QuickStart] - Servicios Web OGC.
  • MapServer - [QuickStart] - Servicios Web OGC.
  • deegree - [QuickStart] - Servicios Web OGC.
  • ncWMS - [QuickStart] - Servicio Web de Mapas.
  • EOxServer - [QuickStart] - Web Coverage Service
  • GeoNetwork - [QuickStart] - Catálogo de Metadatos.
  • pycsw - [QuickStart] - Catálogo de metadatos.
  • MapProxy - [QuickStart] - Proxy WMS y servicios de teselas.
  • QGIS Server - [QuickStart] - Servicio WMS.
  • istSOS - [QuickStart] - Servicios de Sensores de Observación (SOS).
  • 52ºNorth SOS - [QuickStart] - Servicios de Sensores de Observación
  • 52°North WPS - [QuickStart] - Servicios Web de Geoprocesamiento.
  • ZOO Project - [QuickStart] - Servicios Web de Geoprocesamiento.
  • PyWPS - [QuickStart] - Web Processing Service
  • TinyOWS - [QuickStart] - Servicio WFS-T.

  • pgRouting - [QuickStart] - Routing para PostGIS.

SIG de campos específicos¶

Datos¶

Conjuntos de datos espaciales:

  • Natural Earth - Conjuntos de datos Geográficos Globales.

  • OpenStreetMap - Extracto de datos de ejemplo de OpenStreetMap.
  • Conjunto de datos NetCDF - Series temporales de temperatura y precipitación.

Bibliotecas geoespaciales¶

  • libLAS - [QuickStart] - libLAS acceso datos LIDAR.
  • Iris - [QuickStart] - Meteorología y climatología.

Otros programas de interés (no instalados)¶

  • Geopublisher - Catálogo, retirado después de OSGeo-Live 6.0.
  • AtlasStyler - Editor de Estilos, retirado después de OSGeo-Live 6.0.
  • 52°North WSS - [QuickStart] - Web Security Service, retirado después de OSGeo-Live 6.5
  • MapFish - [QuickStart] - Framework de Web Mapping, retirado después de OSGeo-Live 7.9.
  • GpsDrive - [QuickStart] - Navegación GPS , retirado después de OSGeo-Live 9.0.
  • TileMill - [QuickStart] - Estilos y publicación, retirado después de OSGeo-Live 9.5.
